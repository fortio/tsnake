# tsnake

Play the classic game of snake in the terminal

## Install
You can get the binary from [releases](https://github.com/fortio/tsnake/releases)

Or just run
```
CGO_ENABLED=0 go install fortio.org/tsnake@latest  # to install (in ~/go/bin typically) or just
CGO_ENABLED=0 go run fortio.org/tsnake@latest  # to run without install
```

or
```
brew install fortio/tap/tsnake
```

or
```
docker run -ti fortio/tsnake
```


## Usage

```
tsnake help

flags:
```
